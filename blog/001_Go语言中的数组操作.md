## 数组初始化

### 示例1

```
var arr [5]int
fmt.Println(arr)
```

输出

```
[0 0 0 0 0]
```

### 示例2

```
arr := [5]int{1, 2, 3, 4, 5}
fmt.Println(arr)
```

输出

```
[1 2 3 4 5]
```

## 数组遍历

```
var arr [5]int
for i := 0; i < len(arr); i++ {
    arr[i] = i
}
fmt.Println(arr)
```

输出

```
[0 1 2 3 4]
```

## 二维数组

```
var arr [3][5]int
for i := 0; i < 3; i++ {
    for j := 0; j < 5; j++ {
        arr[i][j] = (i + 1) * (j + 1)
    }
}
fmt.Println(arr)
```

输出

```
[[1 2 3 4 5] [2 4 6 8 10] [3 6 9 12 15]]
```
